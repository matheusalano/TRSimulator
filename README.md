# Toy Robot Simulator

## How to build
1. Open `TRS.xcodeproj`
2. Select `TRS` target.
3. Run.

## How to run with `test_data.txt`
1. Run.
2. Select `File` option.
3. Enter the file's full path.
4. Check expected output (`1,4,NORTH`).
